# 🛒 STORELANDIA - Full-Stack E-commerce API

Welcome to **STORELANDIA**! This is my first-ever API project, built to explore the world of web development, backend logic, and security. 

---

## 🚀 Project Status: In Early Development
Currently focused on: **The Authentication System (Login & Register)**.

## 🧠 What I'm Learning with this Project

This project is a journey of discovery. I'm not just writing code; I'm learning how to build a real system:
- **AI-Enhanced Learning:** I use AI as a coding partner and mentor. While some parts of the JavaScript and Backend logic were developed with the help of AI to speed up the process, I make it a priority to **fully understand the data flow and logic** before moving forward.
- **JWT (JSON Web Tokens):** Learning how to manage user sessions and secure routes using tokens.
- **API Architecture:** Discovering how the Frontend (HTML/JS) communicates with the Backend (Flask/Python) using JSON and `fetch`.
- **UI/UX Design:** Creating a modern "Glassmorphism" interface using Bootstrap 5 and custom CSS.
- **Problem Solving:** Debugging real-world issues like ID conflicts and DOM manipulation.

## 🛠️ Tech Stack

- **Development Environment:** Visual Studio Code (VS Code).
- **Backend:** Python with **Flask**.
- **Security:** JWT (JSON Web Tokens).
- **Frontend:** HTML5, CSS3, and Vanilla JavaScript.
- **OS:** Developed on **Fedora Linux**.

## 📌 Features (Roadmap)

- [x] **Dynamic Auth Interface:** Single-page toggle between Login and Register.
- [ ] **Secure Login:** Validating users and generating JWT tokens.
- [ ] **User Roles:** Distinct permissions for `Sellers` and `Customers`.
- [ ] **Product CRUD:** API routes to list and manage store items.
- [ ] **Data Persistence:** Using JSON files for management (scaling to SQL in the future).

## 🔧 How to Run

1. Clone the repository:
   ```bash
   git clone [https://github.com/MiguelLuizCarvalho/storelandia.git](https://github.com/MiguelLuizCarvalho/storelandia.git)

2. Install dependencies: 
    pip install flask pyjwt

3. Start the flask server:
    python app.py

4. Open your browser at: http://127.0.0.1:5000
